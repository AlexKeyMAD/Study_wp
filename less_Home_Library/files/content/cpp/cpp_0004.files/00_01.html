ПРЕДИСЛОВИЕ

Разработка приложений приобрела огромную популярность из-за огромного влияния имеет на различных секторах. На этом быстро развивающемся рынке стало критически важно иметь правильный набор инструменты, позволяющие разработчикам создавать практичные, удобные и эффективные приложения. Эта книга ориентирована на использование и реализацию Small Windows, объектно ориентированной на C ++ библиотека классов, которая упрощает разработку интерактивных приложений Windows.

О чем эта книга

Глава 1. Введение, дает введение в Small Windows, которая представляет собой библиотеку классов, которая инкапсулирует часть Win32 API.

Глава 2. Hello, Small World! Начинается с создания (очень) небольшого приложения - Small Версия для Windows известной программы Hello World. Затем мы продолжим (все еще довольно маленькое) приложение, которое обрабатывает круги в окне. Пользователь может добавлять и перемещать круги, изменить их цвет, а также сохранить и загрузить круги.

В главе 3 «Создание приложения для игры в тетрис» рассматривается версия классической игры «Тетрис». Семь с экрана падают разные фигуры, и задача пользователя - перемещать или вращать их, чтобы можно было полностью заполнить и удалить как можно больше строк.

Глава 4. Работа с фигурами и фигурами, учит, как создать программу для рисования. который можно рассматривать как более продвинутую версию приложения Circle. Возможно создавать и удалять фигуры, а также отмечать и перетаскивать фигуры.

В главе 5 «Иерархия фигур» продолжается построение программы рисования. Мы можем определить иерархия классов с линиями, стрелками, прямоугольниками и эллипсами.

Глава 6. Создание текстового процессора, описывает текстовый процессор, способный к форматированию. отдельные персонажи.

Глава 7. Ввод с клавиатуры и вычисление символов, обсуждает, как текстовый процессор обрабатывает множество комбинаций ввода с клавиатуры и вычисляет размер и положение каждой индивидуальный характер.

В главе 8 «Создание приложения для работы с электронными таблицами» рассказывается о конечном приложении, которое программа для работы с электронными таблицами, способная вычислять формулы по четырем правилам арифметики. это также возможно вырезать и вставлять блоки ячеек.

Глава 9. Интерпретация формул, объясняет, что когда пользователь вводит формулу, нам нужно интерпретировать это. Процесс делится на сканирование и разбор, которые мы рассмотрим в эта глава.

В главе 10 «Фреймворк» описывается основная часть Small Windows. Эта глава начинается описание Small Windows. Класс Application обрабатывает цикл сообщений приложения и регистрации классов Windows. Класс Window обрабатывает основные оконная функциональность.

Глава 11. Документ, говорит о подклассах Window на основе документов, то есть о Класс документа, который обеспечивает базовые функции документа, такие как меню и ускорители и стандартную платформу документов, которая обеспечивает рамки.

Глава 12. Вспомогательные классы, исследует набор небольших точек обработки вспомогательных классов. и размеры, прямоугольники, цвета и шрифты, динамические списки и древовидные структуры.

Глава 13. Реестр, буфер обмена, стандартные диалоги и предварительный просмотр, объясняет реализация реестра и буфера обмена, стандартные диалоги сохранения и загрузки файлов, выбор цвета или шрифта или печать документа. В этой главе также объясняется реализация класса для предварительного просмотра печати.

В главе 14. Диалоги, элементы управления и настройка печати, описывается возможность создания пользовательских диалоговые окна с элементами управления, такими как кнопки, флажки, переключатели, списки, комбо коробки и текстовое поле. Ввод текстового поля можно преобразовать в любой тип. Наконец, печать Диалог установки - это настраиваемый диалог, снабженный соответствующими элементами управления.

Что вам понадобится для этой книги

Прежде всего, вам необходимо скачать Visual Studio на свой компьютер. Предлагаю вам скачать и установите Express для рабочего стола, который является бесплатным и доступен по адресу

Затем есть два способа установить Small Windows:

  1. Если вы хотите следовать структуре глав этой книги, вы можете скачать ее с . Это сделано созданием наборов проектов Visual Studio, содержащих приложения из этой книги.
  2. Если вам нужен весь код в одном решении Visual Studio, вы можете загрузить C ++ Решение для программирования Windows в файле программирования Windows Cpp.
  3. Если вы хотите писать код с помощью Small Windows самостоятельно, вы можете скачать пустой проект в файле Empty Project. Это приложение, содержащее только небольшой исходный код Windows с очень простым приложением. Вы можете изменить имя проекта и добавьте свой собственный код для конкретного приложения.

Для кого эта книга

Эта книга предназначена для разработчиков приложений, которым нужен быстрый подход к Windows программированию. Он научит вас, как разработать объектно-ориентированную библиотеку классов на C ++ и расширенные приложения в Windows. Базовые знания C ++ и объектно-ориентированного программирования. Framework призван максимально использовать возможности этой книги.
