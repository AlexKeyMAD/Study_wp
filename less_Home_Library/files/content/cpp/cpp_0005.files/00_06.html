Синтаксис SQL

Этот документ описывает язык SQL, который понимает SQLite.

SQL в понимании SQLite

SQLite понимает большую часть стандартного языка SQL. Но он опускает некоторые функции, но в то же время добавляет несколько собственных функций. В этом документе делается попытка точно описать, какие части языка SQL SQLite поддерживает и не поддерживает. Также предоставляется список ключевых слов SQL. Синтаксис языка SQL описывается синтаксическими диаграммами.
